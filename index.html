How to use GitHub to Create Live Websites

Step 1 - Create Repository

Login to your github account and click the plus button at the top right. Then click "New repository."

Step 2 - Name Repository

Name the repository next to where the owner is specified. Optionally, select "Initilize this repository with a README" then click "Create repository."

Step 3 - Upload Files

Once you are redirected (after the repository is done being created) drag and drop files onto the page to begin uploading. Alternatively, you can manually search for files to upload.
Once those have uploaded, optionally enter a description of the commit and press the "Commit changes" button.

Step 4 - Navigate to Settings

Wait for the file processing screen to disappear. Now that the files are committed, navigate to the settings page using the toolbar.

Step 5 - Select Master Branch

Once on the settings page, navigate down to the GitHub Pages section. Click "None" and select "master branch" to tell GitHub what folder to host from.

Step 6 - Wait

Now you wait. You can see the link to your site, but GitHub is currently in the process of publishing it. Hang on.

Step 7 - Click It!

Once your site has been published, you will see the section with your link has changed to a green background. Click that link like there's no tomorrow. The end.
